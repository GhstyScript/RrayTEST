Ghsty Assistant

INTELIGENCIA ARTIFICIAL — BY GHSTYAPPS
⚡ Versión 1.0 — Major Release 2025

Ghsty Assistant es una inteligencia artificial con personalidad propia desarrollada por GhstyApps. Con sistema de usuarios, roles, seguridad avanzada y memoria de conversaciones. El primer gran proyecto de Rrayner, construido desde cero junto a Claude.

⚡ CARACTERÍSTICAS

🤖

Personalidad Propia

Ghsty Assistant tiene identidad única desarrollada por GhstyApps. No es un chatbot genérico.

🌐

Multilingüe

Responde en 5 idiomas: Español, English, Français, Português y Deutsch.

👥

Sistema de Usuarios

Registro e inicio de sesión con contraseñas encriptadas. Roles de Admin y Usuario.

👑

Panel de Administración

Control total de usuarios, bloqueos, sanciones, anuncios y historial de conversaciones.

🧠

Memoria de Conversaciones

Ghsty Assistant recuerda conversaciones anteriores de cada usuario.

🔒

Seguridad Avanzada

Sistema de detección de actividad sospechosa con alertas en tiempo real para el admin.

🎨

Nueva Interfaz

Diseño moderno con PyQt6, barra lateral, animaciones y estilo oscuro profesional.

💾

Guarda Conversaciones

Exporta cualquier conversación a un archivo de texto con fecha y hora automática.

🔔

Notificación Sonora

Sonido de notificación al recibir cada respuesta.

📋 HISTORIAL DE VERSIONES

v0.1 — LANZAMIENTO INICIAL

Primera ventana con chat funcional. El momento en que todo comenzó.

v0.2 — EXPANSIÓN DE CAPACIDADES

Limpiar chat, Enter para enviar y guardado de conversaciones.

v0.3 — IDENTIDAD Y MULTILINGÜISMO

Nombre G-P01, personalidad propia y selector de 5 idiomas.

v0.4 — CONSCIENCIA EN TIEMPO REAL

Indicador de escritura, contador de mensajes y threading.

v0.5 — MEMORIA DE DESARROLLO

La IA adquiere consciencia de su propio historial de versiones.

v0.6 — ACCESO RESTRINGIDO

Pantalla de login con contraseña y límite de intentos.

v0.7 — IDENTIDAD VISUAL

Imagen oficial generada con IA integrada en la interfaz.

v0.8 — NOTIFICACIÓN SONORA

Sonido de notificación al recibir respuestas.

v0.9 — SISTEMA DE USUARIOS

Base de datos SQLite, roles Admin y Usuario, contraseñas encriptadas con bcrypt.

v1.0 — MAJOR RELEASE ← ACTUAL

Renombrado a Ghsty Assistant. Nueva interfaz PyQt6 con barra lateral, panel admin completo con bloqueos, sanciones, anuncios, historial de chats y sistema de seguridad.
